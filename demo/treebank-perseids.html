Case A

Text with data-alpheios_tb_doc and data-alpheios_tb_sent attributes (will use findWord to get a wordId):

περὶ πολλοῦ ἂν ποιησαίμην

εὖ γὰρ οἶδʼ ὅτι

Case B

Text with data-alpheios_tb_doc, data-alpheios_tb_sent, and data-alpheios_tb_word attributes:

περὶ πολλοῦ ἂν ποιησαίμην

εὖ γὰρ οἶδʼ ὅτι

Case C

Text with data-alpheios_tb_ref attributes (a legacy formatting):

περὶ πολλοῦ ἂν ποιησαίμην

εὖ γὰρ οἶδʼ ὅτι

Case D

A latin text with a word primaque that returns multiple entries from a treebank (uses data-alpheios_tb_word attributes):

adspirate meis primaque ab origine mundi

Case E

A latin text with a word primaque that returns multiple entries from a treebank (uses data-alpheios_tb_ref attributes):

adspirate meis primaque ab origine mundi

Error handling

A text with incorrect markup

adspirate (data-alpheios_tb_word is missing)
meis (data-alpheios_tb_word has incorrect value)
primaque (unparsable value in data-alpheios_tb_word)
ab (incorrect value in data-alpheios_tb_doc)
 origine mundi

Text which will fail requests due to invalid document

adspirate meis primaque
