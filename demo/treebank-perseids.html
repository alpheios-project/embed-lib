Case B

Text with data-alpheios_tb_doc, data-alpheios_tb_sent, and data-alpheios_tb_word attributes:

περὶ πολλοῦ ἂν ποιησαίμην

εὖ γὰρ οἶδʼ ὅτι

Case C

Text with data-alpheios_tb_ref attributes (a legacy formatting):

Case D

A latin text with a word primaque that returns multiple entries from a treebank (uses data-alpheios_tb_word attributes):

Case E

A latin text with a word primaque that returns multiple entries from a treebank (uses data-alpheios_tb_ref attributes):

adspirate meis primaque ab origine mundi

Case F

Disambiguation tests:

A word has two lexemes returned from a morph adapter. A first lexeme has some inflections. A treebank data returns a lexeme that is matching a first lexeme from a morph adapter by word and part of speech. Treebank's lexeme has no inflecitons. First lexeme in a homonym shown in a popup should be marked as disambiguated: παρ᾽

A word that is contained in a treebank only, not in a morph analyzer. It should be marked as disambiguated: φθίνωσιν
